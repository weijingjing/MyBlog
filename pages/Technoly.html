元素的具体讲解

  1. eg:meta charset="UTF-8" 表示字符集是utf-8支持展示中英文和一些特殊字符
  2. eg:meta name="viewport" content="width=device-width, initial-scale=1", maximum-scale=1, user-scalable=no 表示屏幕自定义展示的大小，最小展示一倍，最大展示一倍，不允许用户缩放
  3. eg:viewport 设定屏幕展示效果，如果不制定为980px iphone标准为375px iphone5为320px,width=device-width展示屏幕的宽度
  4. eg:base href="/" 表示基础链接，a标签如果是123,那链接到123/下面的文件

元素主要的特性

a[href,target]
href指定链接，target打开方式eg:_blank新窗口打开
img[src,alt]
src指定路径，alt表示题目指示语
table td[colspan,rowspan]
colspan上下合并单元格,rowspan左右合并单元格
form[target,method,enctype]
target规定在何处打开url_blank _self _parent _top framename method发送form-data的HTTP方法（post,get） enctype发送数据前对其进行编码，主要针对post请求
input[type,value]
type类型，value表单的默认值
button[type]
type类型:submit reset delete
select>option[value]
单选option value值，用id作唯一识别
label[for]
标签eg:[label for="SSN">Social Security Number: input type="text" name="SocSecNum" id="SSN" />]

HTML面试真题

  1. doctype的意义是什么？
    • 让浏览器以标准模式渲染
    • 让浏览器知道元素的合法性
    • DOCTYPE标签是一种标准通用标记语言的文档类型声明，它的目的是要告诉标准通用标记语言解析器，它应该使用什么样的文档类型定义（DTD）来解析文档。
  2. HTML XHTML HTML5的关系？
    • HTML属于SGML
    • XHTML属于XML，是HTML进行XML严格化的结果
    • HTML5不属于SGML或XML，比XHTML宽松
  3. HTML5有什么变化？
    • 新的语义话元素
    • 表单增强
    • 新的API（离线/音视频/图形/实时通信/本地存储/设备能力）。
    • 分类和嵌套变更
  4. em和i有什么区别？
    • em是语义化的标签，表强调
    • i是纯样式的标签，表斜体
    • HTML5中i不推荐使用，一般用作图标。
  5. 语义化的意义是什么？
    • 开发者容易理解
    • 机器容易理解结构（搜索/读屏软件）
    • 有助于SEO
    • semantic microdata在html中加入标签
  6. 哪些元素可以自闭合？
    • 表单元素input
    • 图片img
    • br/hr
    • meta link
  7. HTML和DOM的关系？
    • HTML是死的
    • DOM是由HTML解析而来的，是活的
    • JS可以解析DOM
  8. property和attribute的区别？
    • attribute是死的
    • property是活的
  9. form的作用有哪些？
    • 直接提交表单
    • 使用submit/reset按钮
    • 便于浏览器保存表单
    • 第三方库可以整体提取值
    • 第三方库可以进行表单验证
